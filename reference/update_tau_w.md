# Update Tau_w Hyperparameter for GP Priors

Update Tau_w Hyperparameter for GP Priors

## Usage

``` r
update_tau_w(lastgammak_all, lastw_all, a_tau_w, b_tau_w)
```

## Arguments

- lastgammak_all:

  Inclusion indicators (matrix).

- lastw_all:

  GP weights (matrix).

- a_tau_w:

  Shape parameter for tau_w prior.

- b_tau_w:

  Rate parameter for tau_w prior.

## Value

Updated tau_w value.
